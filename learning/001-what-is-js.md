# JavaScript - Introduction

## What is JavaScript?

JavaScript is a high-level, interpreted, dynamically typed programming language used to make web pages interactive. It can run in browsers and on servers using Node.js.

---

## Website Components

| Technology | Purpose   |
| ---------- | --------- |
| HTML       | Structure |
| CSS        | Styling   |
| JavaScript | Behavior  |

---

## Why JavaScript?

Without JavaScript:

- Static websites
- No button actions
- No form validation
- No animations
- No dynamic content

With JavaScript:

- Interactive UI
- Form validation
- API calls
- Animations
- Dynamic updates

---

## Where JavaScript Runs

- Browser
- Node.js (Server)

---

## Uses of JavaScript

- Frontend Development
- Backend Development
- Mobile Apps
- Desktop Apps
- Games
- APIs

---

## Execution Flow

User Action

↓

Browser detects event

↓

JavaScript executes

↓

HTML/CSS updates

↓

User sees result

## Example

```javascript
let name = "Yesu";

console.log("Hello " + name);
```

Output

```
Hello Yesu
```

## Interview Definition

JavaScript is a high-level, interpreted, dynamically typed programming language that adds interactivity to web pages. It runs in browsers and on servers using Node.js and is widely used for frontend and backend development.
